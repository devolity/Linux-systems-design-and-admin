Gloucester's Favorites Collector (GFC)
--------------------------------------

Quick Install
-------------
1- Create a user and database in mysql.
2- Unpack this source tree into the desired location.
3- Edit the file .gfcrc, inserting the user name, password, and database
   name where indicated.
4- Load the page install.php in a browser.
5- Delete the install.php file.
6- Use the site.
